Agent Workflow Diary

记录人类把接力棒交给 AI 的日常证据

这里按日期记录我与 AI Agent 的真实协作片段。每篇日记都围绕一个问题：人类把什么经验、流程、工具或行动权交给了 AI，AI 接住了什么，人类又保留了什么判断、边界和责任。

先看场景

截图不是证据堆砌，先说明当天要解决什么问题、为什么需要 Agent 参与。

再看互动

记录 Agent 的拆解、工具调用、追问、错误恢复，以及人类如何接管关键判断。

最后看复盘

每篇都要沉淀一个可复用经验，或说明一个失败边界、幻觉风险、上下文管理问题。
