Human-to-AI Handoff Manifesto

记录人类把创造、判断与行动的接力棒交给 AI

这不是一个聊天截图相册，而是一份正在发生的人机交接观察笔记：当人类把经验写成上下文，把流程交给工具，把重复行动交给 Agent，我们如何重新定义创造、责任、边界、情感和判断。

Project Threads

支撑这些工作流的项目线索

公开项目用于展示工程能力，私有项目只做脱敏功能说明。

01

MirrorAgent

Android-first 主动型个人 Agent 助理，围绕 Capture、Memory、Discovery 和 Assistance 探索长期陪伴与低噪声主动发现。

02

mirror-android

Android 设备感知与统计底座，包含前台服务、UsageEvents、权限引导、本地快照和 Top App 分析。

03

svmes-context

Agent 方法论与长期上下文仓库，按 Memory、Skill、Harness、Experience 组织判断标准、技能、结构和复盘。

04

TendrilFlow

本地编码 Agent 任务协作工作台，关注多 Agent 协作、trace、handoff、review 和可观察性。

05

agent-harness-benchmark

多语言 Agent 工程能力测试题库，关注代码理解、多文件修改、回归修复、测试补充和过程质量。

06

Yamen

轻量多 Agent 协作架构探索，用角色治理、升级机制和复核触发降低编排成本。

Safety & Boundary

截图和案例的脱敏原则

页面会尽量展示方法，不展示敏感内容。每个案例进入作品集前都需要过一遍边界检查。

不放密钥 隐藏 token、API Key、cookie、私有链接和账号信息。
不放公司代码 业务代码、客户数据、内部路径和未公开需求全部脱敏。
不放隐私对话 个人情绪、他人信息和未授权聊天记录不进入页面。
保留判断过程 重点展示任务目标、Agent 行为、人类复核和边界经验。
